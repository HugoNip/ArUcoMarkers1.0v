# ArUco Markers Detection

## Apparatus
webcam (monocular)

## OpenCV Version
OpenCV 4.3.0  
OpenCV Contrib 4.3.0

## Steps to build
To compile on Linux, mac or Windows using Mingw
```
mkdir build
cd build
cmake ..
make
```

## Executables
The following applications are generated.

./basicMarkerDetection  
./markerCreation  
./poseEstimation
